# Security Policy

## Supported Scope

This repository is the open-source release-discovery companion for SafeCLI.

Radar watches package-registry releases, scores candidates, and invokes a local
SafeCLI installation. It should not be treated as a replacement for SafeCLI's
scanner, verdict cache, or policy engine.

## Reporting a Vulnerability

If you discover a security vulnerability:

1. Do not disclose it publicly first.
2. Contact project maintainers privately through repository maintainers/channels.
3. Include clear reproduction steps, impact, and suggested mitigation if available.

We will acknowledge reports quickly, investigate, and coordinate a fix and
disclosure timeline.

## Disclosure Expectations

- We request responsible disclosure.
- We will credit reporters when appropriate and with consent.
